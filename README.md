# Student Registration Form

This project is a Student Registration Form developed using HTML and CSS. It demonstrates the use of form elements such as input fields, radio buttons, checkboxes, textareas, and file upload along with basic CSS styling for layout and design.

## Features
- Collects student personal information
- Gender selection using radio buttons
- Address and hobbies text areas
- File upload for student photo
- Declaration checkbox
- Clean and simple UI design

## Technologies Used
- HTML5
- CSS3
